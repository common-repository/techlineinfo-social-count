A plugin to display social share icons and counter just as Mashable website

== Description ==

This plugin displays a social share counter and sharing icons to share your blog posts on Twitter, Google Plus, Pinterest, Linked-in and Stumbleupon just as seen in the popular website Mashable.
Jquerry plugin from Sharerrr.com is utilized for displaying the share count. 

= Features =
* No images used- All social media icons are fontawesome fonts.
* Social Share Counter. 
* Options to enable/disable the plugin in Posts and Pages.
* Short Code and  code to Embed the Plugin in Theme.
* Responsive design.
* Social Share Sticky Top Bar
Please visit the <a href="http://www.techlineinfo.com/mashable-inspired-social-count-and-share-plugin-for-wordpress">Plugin Support Page</a> for More details and support. 
== Installation ==

* Upload the plugin folder to the /wp-content/plugins/ folder of your WordPress installation.  
* Activate the plugin.   
* Plugin settings page can be accessed through the "Techline Social" menu Link
